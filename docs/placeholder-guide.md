# プレースホルダー（入力欄）の書き方

Zero-Err では、テンプレート本文の「入力してもらいたい部分」を **プレースホルダー** で指定します。

## 書式

```
{{キー名}}
```

- **二重波かっこ** `{{` と `}}` で囲みます。
- **キー名** は、半角英数字がおすすめです（日本語も使えます）。
- キー名にはスペースを入れないでください。

## 例

| 本文の書き方 | 説明 |
|-------------|------|
| `到着が{{分数}}分遅れます` | 「分数」という入力欄が1つできる |
| `{{駅名}}におります` | 「駅名」という入力欄が1つできる |
| `{{理由}}の影響で` | 「理由」という入力欄が1つできる |

## 入力項目との対応

1. **テンプレートを追加** するとき、「本文」に `{{キー名}}` を書きます。
2. 同じ画面の「入力項目」で、**キー** に同じ名前（例：`分数`）を入れます。
3. **表示名** は、利用者に見せるラベル（例：「遅延分数」）を入れます。
4. テンプレートを使う画面では、この表示名の入力欄が表示され、入力値が本文の `{{キー名}}` と置き換わります。

**ポイント:** 本文の `{{〇〇}}` の「〇〇」と、入力項目の「キー」は **同じ文字列** にしてください。

## ワード・エクセル・ドキュメント・スプレッドシートから追加する場合

1. もとの文書（Word / Excel / Google ドキュメント / スプレッドシートなど）で文章をコピーします。
2. Zero-Err の「テンプレートを追加」の **本文** に貼り付けます。
3. 入力欄にしたい部分を、手で `{{キー名}}` に書き換えます。
4. 「本文から入力項目を自動抽出」ボタンを押すと、`{{キー名}}` から入力項目が自動作成されます。あとは表示名や種類（テキスト・日付・数値）を必要に応じて編集してください。
