# 自分のテンプレートを追加する方法

Zero-Err で「テンプレートを追加」から、自分用の文書テンプレートを登録する手順です。

---

## 事前準備：ログイン

テンプレートを保存するには **ログイン** が必要です。

1. 画面上部の **「新規登録」** でアカウントを作成（まだの場合）
2. **「ログイン」** でメールとパスワードを入力してログイン

---

## テンプレートを追加する手順

### 1. 「テンプレートを追加」を開く

- ホーム画面で **「テンプレートを追加」** のカードをクリック  
  → テンプレート追加の画面が開きます。

### 2. テンプレート名を入力

- **テンプレート名** に、自分が分かりやすい名前を入れます。  
  例：`電車遅延の連絡` `休暇申請` `報告書`

### 3. 本文を書く（ここが一番重要）

- **本文** に、完成形の文章を書きます。
- **入力してもらいたい部分だけ**、次の形で囲みます。

  ```
  {{キー名}}
  ```

  **例：**

  ```
  お疲れ様です。
  到着が{{分数}}分ほど遅れる見込みです。{{駅名}}付近で{{理由}}の影響が出ております。
  よろしくお願いいたします。
  ```

  - `{{分数}}` → あとで「分数」用の入力欄ができる
  - `{{駅名}}` → 「駅名」用の入力欄
  - `{{理由}}` → 「理由」用の入力欄

- ワード・エクセル・Googleドキュメントなどに既にある文章は、**コピーして本文に貼り付け**し、入力してほしい部分だけを `{{キー名}}` に書き換えても大丈夫です。

### 4. 「本文から入力項目を自動抽出」を使う（おすすめ）

- 本文に `{{〇〇}}` を書いたあと、**「本文から入力項目を自動抽出」** ボタンを押します。
- すると、`{{分数}}` `{{駅名}}` `{{理由}}` などから、**入力項目が自動で一覧に追加**されます。
- あとで「表示名」や「種類」だけ、必要に応じて編集すればOKです。

### 5. 入力項目を整える

- 一覧に出た各項目で、次を設定します。
  - **キー** … 本文の `{{〇〇}}` の「〇〇」と **同じ文字** にします（自動抽出で既に入っているはず）。
  - **表示名** … 使う人に見せるラベル。例：`分数` → `遅延分数（分）`
  - **種類** … **テキスト** / **日付** / **数値** のどれか。
    - 日付にすると、選択した日付が自動で「○月○日(金)」のように表示されます。
  - **必須** … 入力必須にしたい項目だけチェック。

### 6. カテゴリ・公開（任意）

- **カテゴリ** … 後で一覧を分けたいとき用。例：`連絡` `申請`
- **他のユーザーにも公開する** … チェックすると、他のログイン用户もこのテンプレートを使えます。

### 7. 保存する

- **「保存する」** をクリックすると、テンプレートが登録されます。
- 成功すると「テンプレートを追加しました。」と表示されます。

---

## 追加したテンプレートの使い方

1. ホームの **「テンプレートを使う」** をクリック
2. 一覧から **追加したテンプレート** を選ぶ
3. ①・②・③… の入力欄に値を入れる（日付はカレンダーから選択可能）
4. プレビューで文章を確認し、**「コピー（番号なしで本文のみ）」** でコピーして使う

---

## ポイントまとめ

| やりたいこと | やり方 |
|-------------|--------|
| 入力欄にしたい部分を指定する | 本文で `{{キー名}}` と書く |
| キー名と入力項目を一致させる | 入力項目の「キー」を、`{{〇〇}}` の「〇〇」と同じにする |
| 日付を「2月16日(金)」のように出す | 入力項目の「種類」で **日付** を選ぶ |
| 既存の文書から作る | 文書をコピーして本文に貼り付け、該当箇所を `{{キー名}}` に書き換え → 「本文から入力項目を自動抽出」 |

詳しいプレースホルダーの書き方は [placeholder-guide.md](./placeholder-guide.md) も参照してください。
